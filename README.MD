# eCommerce Backend

This Node application the backend for an ecommerce website.

The application gives yout the following options:

- View all items by product, category, id.
- Add new products
- Delete an existing product

## Technologies Used
* Javascript
* Node.js
* NPM
* NPM mySQL2
* NPM Sequelize
* NPM bcrypt
* NPM dotenv
* NPM Express
* Insomnia
